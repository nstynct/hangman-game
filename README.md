Игра Виселица



Игра на Python. Компьютер загадывает слово, игрок отгадывает его по буквам.



Правила

\- Компьютер загадывает случайное слово

\- У игрока 6 попыток

\- За угаданную букву: +100 очков

\- За неверную букву: -50 очков

\- За победу: +500 бонусных очков



Как запустить

1\. Открыть репозиторий: git clone https://github.com/nstynct/hangman-game.git

2\. Перейти в папку: cd hangman-game

3\. Запустить игру: py main.py
